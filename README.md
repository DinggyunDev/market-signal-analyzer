# Market Signal Analyzer

주식의 기술적 지표를 분석하고 시장 신호를 요약해주는 도구입니다.

한국 주식과 미국 주식을 티커만 입력하면 자동으로 분석합니다.

## 주요 기능

- **8개 핵심 지표 자동 분석**: VWAP, 거래량, 모멘텀, 스토캐스틱, 이격도, MACD, RSI, OBV
- **종합 시그널 판정**: 지표를 종합하여 강세/약세/과매수/과매도 상태를 자동 해석
- **한국 주식 자동 검색**: 6자리 종목코드 입력 시 코스피/코스닥 자동 검색
- **다종목 비교**: 쉼표로 구분하면 여러 종목을 한눈에 비교
- **워치리스트**: 관심 종목을 저장해두고 한 번에 분석
- **장 상태 자동 감지**: 한국/미국 모두 장중에는 확정된 데이터만 사용

## 분석 지표

| 지표 | 설명 | 아는 게 좋은 이유 |
|------|------|------------------|
| VWAP | 거래량을 고려한 평균가 | 오늘 산 사람들의 평균 매입가를 알 수 있음 |
| 거래량 비율 | 평소 대비 거래 활발도 | 시장이 이 종목에 관심이 있는지 확인 |
| 모멘텀 | 10일 전 대비 가격 변화 | 지금 오르는 힘이 있는지 판단 |
| 스토캐스틱 | 최근 가격 위치 (0~100) | 단기적으로 비싼 건지 싼 건지 확인 |
| 이격도 | 20일 평균과의 거리 | 평균에서 너무 멀어졌는지 확인 |
| MACD | 추세 방향 지표 | 상승/하락 추세의 전환 시점을 포착 |
| RSI | 상대 강도 지수 (0~100) | 과매수(70↑) / 과매도(30↓) 판단 |
| OBV | 거래량 누적 추세 | 큰 손(기관/외국인)의 자금 방향 추정 |

## 실행 방법

이 프로그램은 두 가지 방법으로 사용할 수 있습니다.

| | EXE 실행 (권장) | 개발자 모드 |
|---|---|---|
| **대상** | 일반 사용자 | Python 사용 가능자 및 기능 확장을 하고싶은 사용자 |

| **업데이트** | 새 EXE 파일 다운로드 | `git pull` |

---

### 방법 1: EXE 파일로 실행 (권장)

가장 간단한 방법입니다. Python을 설치할 필요가 없습니다.

1. [Releases 페이지](https://github.com/DinggyunDev/market-signal-analyzer/releases)에서 최신 `MarketAnalyzer.exe`를 다운로드합니다.
2. 다운로드한 파일을 원하는 폴더에 옮깁니다.
3. `MarketAnalyzer.exe`를 더블클릭하면 바로 실행됩니다.

> **참고**: Windows 보안 경고가 뜨면 "추가 정보" → "실행"을 클릭하세요.

---

### 방법 2: 소스코드로 실행 (개발자용)

직접 코드를 수정하거나 최신 버전을 바로 반영하고 싶을 때 사용합니다.

#### 1단계: Git 설치

Git이 설치되어 있지 않다면 먼저 설치합니다.

1. [git-scm.com/downloads](https://git-scm.com/downloads)에서 다운로드합니다.
2. 설치 시 모든 옵션은 **기본값(Next)**으로 진행하면 됩니다.
3. 설치가 끝나면 명령 프롬프트(또는 PowerShell)를 열어 확인합니다:

```
git --version
```
`git version 2.x.x` 같은 메시지가 나오면 성공입니다.

#### 2단계: Python 설치

1. [python.org/downloads](https://www.python.org/downloads/)에서 최신 버전을 다운로드합니다.
2. **설치 시 "Add Python to PATH" 체크박스를 반드시 체크**하세요.
3. 설치 후 확인:

```
python --version
```

#### 3단계: 프로젝트 다운로드 (클론)

명령 프롬프트를 열고 원하는 폴더로 이동한 뒤:

```
git clone https://github.com/DinggyunDev/market-signal-analyzer.git
cd market-signal-analyzer
```

이렇게 하면 프로젝트 폴더가 자동으로 생성됩니다.

#### 4단계: 라이브러리 설치

```
pip install -r requirements.txt
```

처음 한 번만 실행하면 됩니다.

#### 5단계: 실행

```
python main.py
```

## 사용법

프로그램을 실행하면 아래와 같은 화면이 나타납니다.

```
==================================================
   시장 신호 분석기 v1.0.0
   Market Signal Analyzer
==================================================

  사용법:
    티커 입력    삼성전자: 005930 / 미국: TSLA
    다종목 비교  AAPL, MSFT, NVDA
    워치리스트   wl add 005930 TSLA  /  wl
    도움말       help
    종료         q 또는 exit

--------------------------------------------------
[02:30] 티커 입력 (help: 도움말):
```

### 종목 분석

종목코드나 티커를 입력하면 상세 분석 결과가 출력됩니다.

```
005930        → 삼성전자 분석
TSLA          → 테슬라 분석
```

### 여러 종목 비교

쉼표로 구분하면 요약 테이블로 한눈에 비교할 수 있습니다.

```
AAPL, MSFT, NVDA, TSLA
```

### 워치리스트

관심 종목을 저장해두면 매번 입력할 필요 없이 한 번에 분석합니다.

| 명령어 | 설명 |
|--------|------|
| `wl add 005930 TSLA` | 종목 추가 (여러 개 동시 가능) |
| `wl list` | 저장된 종목 목록 보기 |
| `wl` | 저장된 종목 전체 일괄 분석 |
| `wl rm TSLA` | 종목 제거 |
| `wl clear` | 전체 초기화 |

워치리스트는 프로그램을 종료해도 유지됩니다.

### 기타 명령어

| 명령어 | 설명 |
|--------|------|
| `help` | 도움말 표시 |
| `clear` | 화면 정리 |
| `q` 또는 `exit` | 프로그램 종료 |

## AI IDE와 바이브 코딩으로 기능 확장하기

코딩을 잘 몰라도 AI 개발 도구(AI IDE)를 활용하면 이 프로그램을 더 강력하게 업그레이드할 수 있습니다. 예를 들어, AI에게 "속도를 더 빠르게 해줘", "새로운 지표를 추가해줘"처럼 말로 요청하면 코드를 직접 수정해주는 방식입니다. 이런 작업 방식을 흔히 바이브 코딩(Vibe Coding)이라고 부릅니다.

### 1. 기본 추천 IDE

처음 시작할 때는 **VS Code** 또는 **Antigravity**를 추천합니다.

- **VS Code**: 가장 널리 쓰이는 개발 도구입니다. 실행, 수정, 디버깅, 파일 관리가 편해서 처음 시작하기 좋습니다.
- **Antigravity**: AI가 코드 수정뿐 아니라 터미널과 브라우저 작업까지 함께 도와주는 도구입니다. AI에게 자연어로 시키는 방식이 편해서 바이브 코딩에 잘 맞습니다.

**추천 기준**
- 안정적이고 익숙한 환경을 원하면 → **VS Code**
- AI에게 더 많이 맡기고 싶다면 → **Antigravity**

### 2. 자주 쓰는 AI IDE / 에이전트 정리

| 도구 | 특징 | 무료/유료 사용 |
|---|---|---|
| **VS Code + GitHub Copilot** | 가장 무난하고 안정적인 조합입니다. | 무료 체험 및 유료 플랜 제공 |
| **Cursor** | AI가 코드 수정과 설명을 매우 잘 도와줍니다. | 무료 플랜 / 유료 플랜 제공 |
| **Windsurf** | 여러 파일을 오가며 작업하는 능력이 좋습니다. | 무료 플랜 / 유료 플랜 제공 |
| **Antigravity** | AI가 코드, 터미널, 브라우저를 함께 다룹니다. | 무료 사용 가능 (시점에 따라 정책 변경 가능) |

#### 어떤 AI를 쓸 수 있나요?
대부분의 AI IDE는 아래와 같은 AI 모델을 연결해서 사용할 수 있습니다.
- **OpenAI 계열 (GPT)**
- **Claude 계열**
- **Gemini 계열**
- 일부 도구는 자체 AI 모델도 지원합니다.

> **쉽게 설명하면**
> IDE는 "작업 공간"이고, AI 모델은 "도와주는 두뇌"라고 생각하면 됩니다.

### 3. AI에게 어떻게 요청하면 좋을까요?

AI에게는 막연하게 요청하기보다, 무엇을 바꾸고 싶은지, 어떤 조건을 유지해야 하는지를 함께 말해주면 더 잘 작동합니다.

#### 좋은 요청 방식
1. 무엇을 바꾸고 싶은지 말합니다.
2. 기존 기능은 유지해달라고 말합니다.
3. 수정 후 테스트도 같이 해달라고 요청합니다.

#### 예시 템플릿
- 이 프로젝트는 CLI 기반 주식 분석기입니다.
- 출력 형식은 유지하면서 성능을 개선해줘.
- 기존 테스트가 깨지지 않도록 수정하고, 수정 후 테스트 코드도 함께 확인해줘.

### 4. 성능 업그레이드 요청 예시

아래처럼 더 구체적으로 요청하면 AI가 더 잘 수정해줍니다.

#### 속도 / 안정성 개선
- "현재 코드에서 가장 느린 부분을 찾아서 더 빠르게 개선해줘"
- "여러 종목을 한 번에 분석할 때 속도가 느려지지 않도록 최적화해줘"
- "에러가 나도 프로그램이 종료되지 않도록 예외 처리 코드를 추가해줘"
- "중복 계산을 줄여서 더 가볍게 동작하도록 수정해줘"
- "불필요한 데이터 요청을 줄이고 캐시를 적용해줘"

#### 기능 추가
- "볼린저 밴드 지표를 추가하고 해석 문구도 함께 만들어줘"
- "분석 결과를 txt 파일로 저장하는 기능을 추가해줘"
- "워치리스트 종목을 한 번에 분석하는 기능을 만들어줘"
- "미국 주식도 시간대에 맞게 안전하게 분석되도록 수정해줘"
- "사용자의 선호에 따라 현재의 CLI 형식을 웹(Web) 인터페이스로도 확장해줘"

#### 보기 좋게 개선
- "출력 결과를 더 깔끔하고 보기 쉽게 정리해줘"
- "중요한 지표는 더 눈에 잘 띄게 표시해줘"
- "종합 해석 문구를 더 전문가처럼 자연스럽게 바꿔줘"

### 5. 무료로 시작하기 좋은 주식 API

이 프로젝트는 기본적으로 Yahoo Finance 같은 무료 데이터 소스를 활용할 수 있지만, 필요하면 다른 무료 API로도 확장할 수 있습니다.

| API | 특징 | 추천 이유 |
|---|---|---|
| **Twelve Data** | 미국 주식 데이터와 기술 지표 제공 | 무료로 시작하기 좋고 사용이 편합니다. |
| **FMP (Financial Modeling Prep)** | 종목 정보와 일봉 데이터 제공 | 무료 호출 수가 비교적 넉넉합니다. |
| **Alpha Vantage** | 주가, 기술 지표, 뉴스 등 제공 | 기능이 다양해서 학습용으로 좋습니다. |

**추천 순서**: Twelve Data > FMP > Alpha Vantage

> **쉽게 설명하면**
> API는 "주식 데이터를 가져오는 창구"입니다. 프로그램은 이 창구를 통해 가격, 거래량, 종목 정보를 받아옵니다.

### 6. AI에게 API 관련 기능을 요청하는 방법

#### 예시 요청

- "무료 API 요청 제한을 넘지 않도록 캐시 기능을 추가해줘"
- "API 호출이 실패하면 기존 데이터로 대체하는 안전 장치를 넣어줘"
- "미국 주식은 미국장 시간 기준으로, 한국 주식은 한국장 시간 기준으로 처리해줘"

#### 추천 요청 템플릿
- 이 프로젝트에 무료 주식 API를 연결하고 싶습니다
- 가능하면 무료 한도 안에서 동작하도록 만들어주고, API 호출 실패 시에도 프로그램이 멈추지 않게 해줘
- 기존 CLI 출력 형식은 유지해줘

### 7. 핵심 정리

이 프로젝트는 단순한 CLI 분석기에서 끝나는 것이 아니라, AI IDE를 활용하면 **속도 개선, 기능 추가, 출력 개선, API 확장, 웹 인터페이스 전환**까지 쉽게 발전시킬 수 있습니다. 즉, 코드를 하나하나 직접 작성하지 않아도 **"무엇을 만들고 싶은지 잘 설명하는 것"** 만으로도 충분히 프로젝트를 확장할 수 있습니다.
